# Voice Assistant

A simple voice assistant built with Python.

## Features

- Voice recognition
- Text to speech
- Open Google
- Open YouTube
- Play songs on YouTube
- Wikipedia search
- Tell current time

## Requirements

- Python 3.x
- SpeechRecognition
- pyttsx3
- pywhatkit
- wikipedia
- sounddevice
- scipy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
